Project Overview

OpenHaus was an online marketplace designed to connect users with unique real estate listings and home-related services. The platform aimed to provide an easy-to-use interface where buyers, sellers, and service providers could interact seamlessly.

Problem Solved:

OpenHaus aimed to streamline the property discovery and service connection process, reducing friction for users looking to engage with real estate services.

My Role

As the UX/UI Designer, I contributed to:

  • Designing the user interface for the web app and website.
  • Creating wireframes and prototypes to define user flows.
  • Ensuring accessibility and usability for a broad audience.
  • Collaborating with developers to implement designs effectively.
  • Iterating on designs based on user feedback.

Challenges & UX/UI Decisions

Biggest Challenges:

  • Simplifying the user journey for buyers and sellers.
  • Balancing aesthetic appeal with functionality.
  • Making the platform fully mobile-responsive.

Key UX/UI Decisions:

Navigation & User Flow

Simplified onboarding and browsing for efficiency.

Search & Filters

Implemented intuitive filters for quick property discovery.

Visual Hierarchy

Used clear typography and whitespace for better readability.

Mobile Optimization

Designed for seamless use across all devices.

Final Outcome & Lessons Learned

OpenHaus successfully launched and gained initial traction before the startup closed.

If I could improve it today:

  • Enhance search filtering for more personalized results.
  • Introduce microinteractions to improve usability.
  • Test different button placements and messages to find the most effective way to guide users toward taking action.

Skills Strengthened

This project helped me strengthen my expertise in:

  • Complex platform design
  • Information architecture
  • User flow optimization
  • Design system creation
  • Web/mobile UI consistency
  • Prototype development

  • User feedback integration

These real estate marketplace design experiences have been invaluable in shaping my approach to UX/UI for complex digital platforms where user satisfaction directly impacts business success.
